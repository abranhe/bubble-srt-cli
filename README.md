# Install

```
npm install bubble-srt-cli
```

# Usage

```console
Usage:
 $ bubble-srt <array>

Options
	-f, --feedback  Send a feedback
	-h, --help      Show help message and close
	-v, --version   View package Version

Example
 $ bubble-s "d h z a r b i"
	[ 'a', 'b', 'd', 'h', 'i', 'r', 'z' ]
 $ bubble-s "3 2 5 25 74"
	[ '2', '25', '3', '5', '74' ]
```
